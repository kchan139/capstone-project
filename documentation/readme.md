# Documentation

This directory covers the `mrunc` runtime, its bundle format, and the environment files that ship with this repo.

## Contents

- `guides/CONTRIBUTION.md` — build, test, and contribution workflow.
- `guides/makefile-usage.md` — active Makefile targets and the direct runtime flow.
- `guides/vsc-usage.md` — SSH forwarding and the OpenVSCode compose template.
- `notes/environment.md` — Terraform, Ansible, and helper scripts under `environment/`.
- `notes/lightweight-sandbox.md` — isolation and hardening behavior in the runtime.
- `notes/minimal-in-mrunc.md` — what “minimal” means in this project.
- `notes/subset-of-oci.md` — the `config.json` shape used by `mrunc`.
- `oci/oci-subset.json` — example `config.json`.
- `oci/runtime-spec.md` — command and runtime reference.
